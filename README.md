## Описание


Phoenix Shield AV — современный антивирусный комплекс, обеспечивающий многоуровневую защиту компьютера от вредоносного ПО.  

**Ключевая особенность** — расширенная защита от руткитов:
- обнаружение скрытых драйверов и процессов;  
- мониторинг системных вызовов на уровне ядра;  
- предотвращение внедрения вредоносного кода в критические компоненты ОС.

## Основные функции

- Реальное время сканирование файлов и процессов.  
- Эвристический анализ подозрительного поведения.  
- Защита от фишинга и сетевых атак.  
- Регулярное обновление вирусных баз (каждые 2 часа).  
- Лёгкий интерфейс с минимальным потреблением ресурсов.  
- Карантин для изолирования подозрительных объектов.

## Установка

1. Скачайте файл `Phoenix_Shield_AV_v4.9.2_Setup.exe`.  
2. Запустите файл от имени администратора.  
3. Следуйте инструкциям установщика.  
4. После установки перезагрузите компьютер (рекомендуется).  
5. Запустите приложение и выполните первое полное сканирование.

## Обновление

Антивирус автоматически проверяет наличие обновлений каждые 2 часа.  

**Для ручного обновления:**  
1. Откройте главное окно Phoenix Shield AV.  
2. Перейдите в раздел **«Настройки» → «Обновления»**.  
3. Нажмите **«Проверить сейчас»**.
